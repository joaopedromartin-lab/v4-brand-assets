06 / Ícones

Sistema de Ícones

Padrão V4: SVG inline com currentColor, traço de 2px. Biblioteca primária Lucide para estáticos. Para hero ou micro-interações de marca, <lottie-player> com .json do LottieFiles.

Fundamentos

Tamanhos canônicos, 24px é o default. Sempre currentColor, nunca cor fixa no SVG. Traço 2px (1.75 a partir de 48px para evitar peso visual).

20px

24px default

Hero, ilustração

Biblioteca: Lucide

~1.600 ícones, MIT, traço 2px uniforme. Carregado via CDN, instanciado com lucide.createIcons(). Uso: <i data-lucide="arrow-right"></i>. Catálogo completo em lucide.dev/icons.
